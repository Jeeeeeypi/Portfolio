Luna’s Secret is a self-initiated web design project created using the Duda Builder platform. The website is designed to showcase a modern, elegant, and visually appealing layout that reflects the brand’s aesthetic and identity. It demonstrates the use of Duda’s drag-and-drop builder to create a fully responsive and user-friendly website without extensive coding. This project highlights skills in web design, layout structuring, and visual storytelling to build an engaging online presence.
