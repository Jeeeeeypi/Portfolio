Earist Student Handbook 2021

EARIST Student Handbook is a thesis project that transforms the traditional student handbook into a mobile application, developed to improve accessibility and convenience for students. The app presents essential school guidelines, policies, and information through a modern, user-friendly, and mobile-responsive interface. This project demonstrates the use of mobile development concepts to create a more efficient and engaging digital solution for students.

Technologies used: HTML,CSS, JavaScript
